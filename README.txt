Sleep/Wake Quest Prototype V5.2 - Thought Bullets

정적 HTML 게임 프로토타입입니다.

조작:
- WASD / 방향키: 이동
- Q: 번개/베개 모드 수동 전환
- Space / 액션 버튼: 현재 모드 액션
- 베개 모드에서는 액션 버튼을 길게 누르기

규칙:
- 졸음구름은 반경 밖으로 도망갑니다. 번개 모드로 연타해서 깨웁니다.
- 생각벌레는 원거리 생각탄을 발사합니다. 생각탄에 맞으면 액션 게이지가 깎이고 콤보가 줄어듭니다.
- 몬스터가 근성이 개입 반경 밖으로 나가면 신호 상실 처리됩니다.
- 잘못된 모드로 액션하면 해당 몬스터 상태가 악화됩니다.
- 초록 아이템은 잠시 자동전환을 켜줍니다.
- 핀 아이템은 가까운 몬스터를 잠시 고정합니다.
