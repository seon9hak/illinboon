Sleep/Wake Quest Prototype V5.4 - Dispatch Signals

정적 HTML 게임 프로토타입입니다.

조작:
- WASD / 방향키: 이동
- Q: 번개/베개 모드 수동 전환
- Space / 액션 버튼: 현재 모드 액션

규칙:
- 평소에는 반경 10km를 감시합니다.
- 랜덤 타이밍에 수면 불일치 신호가 발생합니다.
- 신호가 발생하면 제한시간 안에 출동해서 처리해야 합니다.
- 졸음구름은 반경 밖으로 도망갑니다. 번개 모드로 연타해서 깨웁니다.
- 생각벌레는 타이밍 바의 중앙 영역에 맞춰 베개 액션을 눌러 진정시킵니다.
- 몬스터가 반경 밖으로 나가거나 제한시간이 지나면 놓친 신호로 기록됩니다.
- 게임오버는 없습니다. 오늘/누적 놓친 신호를 결과로 표시합니다.
